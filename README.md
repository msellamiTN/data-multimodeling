# Modélisation multidimensionnelle et entrepôts de données (L3)

Ce dépôt contient le cours (Markdown), les TD et un mini-projet. Tous les schémas doivent être fournis en Mermaid ou en SVG inline, conformément aux consignes de `command.md`.

## Structure

- `cours/` : chapitres A–E

- `td/` : TD1 à TD3

- `mini-projet/` : énoncé, ressources, évaluation

- `command.md` : consignes de production du support

## Attendus généraux

- Langue : français

- Niveau : L3

- Format : Markdown structuré (titres, tableaux, listes)

- Illustrations : Mermaid/SVG uniquement (pas d’images externes, pas d’ASCII)
