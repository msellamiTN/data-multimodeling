# Grille d’évaluation — Mini-projet BI (L3)

| Critère | Description | Points |
| --- | --- | --- |
| Schéma dimensionnel | Étoile/flocon limité, granularité claire, clés conformes | 4 |
| SQL tables + données | DDL correct, inserts cohérents, intégrité minimale | 3 |
| Requêtes OLAP | Exactitude roll-up/drill-down/top-N, lisibilité | 3 |
| Qualité/Gouvernance | Règles QC pertinentes, prise en compte des risques | 3 |
| Note d’architecture | Choix ROLAP/MOLAP/HOLAP + plan d’actualisation argumenté | 2 |
| Clarté du livrable | Structure, concision, français soigné | 2 |
| Total | 17 |

## Barème d’appréciation

- ≥ 15 : Excellent (maîtrise des concepts et de la mise en œuvre)
- 13–14 : Très bien (quelques points perfectibles)
- 11–12 : Bien (base solide, approfondir performance/gouvernance)
- 9–10 : Acceptable (quelques manques techniques)
- < 9 : Insuffisant (schéma ou requêtes incorrects)
